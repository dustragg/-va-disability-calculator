What areas are affecting you?

Select one or more broad body areas or symptom groups to focus the condition modules below. Choose Show All Conditions if you want to review every mapped condition. This is only a UI filter and does not change ratings, saved values, evidence, planning data, reports, or combined-rating math.

Respiratory conditions are not yet implemented. Consider using a tracking-only custom condition.

How to use this estimator

  1. Select answers that best match the evidence for each condition.
  2. Leave a condition at “not claimed / no compensable symptoms” if it should not be included.
  3. Review each rationale and the combined-rating calculation. Bilateral factor and special monthly compensation are not included in this version; Version 3 adds hip, knee, mental-health, and additional peripheral-nerve templates while preserving the existing combined-rating calculation.
